MOBILE APPLICATION



In order to create a mobile application which will allow the user to watch the current position of ball simultaneously and get the coordinates information,Flutter is used on Android Studio platform.Flutter is an open-source UI software development kit created by Google. So far,random coordinates are generated each second to simulate the app without getting in touch with the actual hardware. The mobile app is kept as simple as possible in order to integrate new updates. The next goal is to integrate the mobile application with hardware and show the identical position of the ball.
